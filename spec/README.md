# 公開仕様（Open Specs）

このディレクトリは、**外部公開して他クライアントに実装を促すための仕様**を置く。
**アプリ本体のコードは非公開（クローズド）**だが、ここに置く**セキュリティ提案の仕様は公開**する。

- 目的: 「不用意なAI露出を業界全体で減らす」提案を、**ベンダー中立**に広める。
- 性質: アプリ内部実装（DB・コマンド・UI）を含めず、**相互運用に必要な“線の上”の取り決めのみ**を記述。
- ライセンス: 公開仕様はオープン（例: CC BY 4.0 / パブリックドメイン相当。確定時に明記）。アプリ本体とは切り離す。

| 仕様 | 内容 |
|---|---|
| [protected-regions-v1.md](protected-regions-v1.md) | 保護領域（プライバシー伏字＋暗号）の相互運用仕様 v1 |

> アプリ内部の設計は `docs/`（非公開）にある。本 `spec/` は**そこから相互運用部分だけを切り出した公開面**。
